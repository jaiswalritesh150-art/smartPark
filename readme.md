# SmartPark 🚗

SmartPark is a real-world parking solution web app that helps users:
- Find nearest parking slots
- Compare prices
- Book parking instantly
- Generate QR-based entry

## Features
- 🔍 Smart Search
- 📍 GPS Location
- 💰 Cheapest & Nearest Sorting
- 🗺 Map View UI
- 📅 Booking System
- 💳 Payment Simulation
- 🎟 QR Entry System
- 📊 Dashboard

## Tech Used
- HTML
- CSS
- JavaScript

## Future Improvements
- Backend Integration (Node.js / Firebase)
- Real-time slot availability
- Google Maps API
- Payment Gateway
